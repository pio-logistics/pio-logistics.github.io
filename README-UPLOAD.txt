PIO LOGISTICS - NETLIFY DEPLOY PACKAGE

Upload the entire PIO-Logistics-Netlify-Ready folder to Netlify Drop.
The website entry file is index.html.

Do not upload index.html by itself because the presentation uses the
presentation-assets folder for images, maps, flags, certificates and fonts.

IPHONE FULL SCREEN
Safari does not allow a normal webpage to remove its address bar by script.
For a true full-screen presentation: open the deployed site in Safari, tap
Share > Add to Home Screen, then launch PIO from the new Home Screen icon.
The top-right full-screen button uses native full screen on supported browsers
and shows these iPhone instructions when Safari blocks the Fullscreen API.
